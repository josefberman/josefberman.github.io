Image Guidelines:
- Use high-quality images with good resolution
- For projects, consider using screenshots of visualizations, models, or relevant graphics
- Keep a consistent style and color scheme across all images
- Dark/blue toned images work best with the site's color scheme
- Add subtle orange elements to complement the site's accent color

Resources for free stock photos and graphics:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)
- Undraw (https://undraw.co) - Great for tech/science illustrations

For project screenshots:
- Use tools like Figma, Canva, or Adobe Creative Suite to create professional thumbnails
- Overlay text or icons on images if needed for clarity
- Consider adding a subtle gradient overlay to maintain visual consistency

Note: For convenient deployment on GitHub Pages, you can use free stock photos or create your own images.
For a professional profile picture, consider using a headshot with a clean background.
