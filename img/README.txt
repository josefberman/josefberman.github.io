Image Placeholders README

This directory should contain the following images:

1. profile-placeholder.jpg - Profile photo for the About section (recommended size: 400x500px)
2. project1.jpg - Thumbnail for Project 1 (recommended size: 800x600px)
3. project2.jpg - Thumbnail for Project 2 (recommended size: 800x600px)
4. project3.jpg - Thumbnail for Project 3 (recommended size: 800x600px)
5. blog1.jpg - Thumbnail for Blog post (recommended size: 800x600px)

You should replace these placeholders with your actual images.

Note: For convenient deployment on GitHub Pages, you can use free stock photos or create your own images.
For a professional profile picture, consider using a headshot with a clean background.

Resources for free stock photos:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Pixabay (https://pixabay.com)
